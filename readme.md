\# 📊 Steel Surcharge Calculator



A modern tool for sales professionals to calculate steel surcharges, create offers, and manage approval workflows.



\---



\## 🚀 Project Overview



\*\*Steel Surcharge Calculator\*\* is a web-based application designed to streamline pricing calculations for steel products. It supports sales teams in creating accurate offers quickly and efficiently.



The application works in real-time — all changes are instantly reflected in the final price without the need to refresh the page.



\---



\## ✨ Features



\### 🧮 Pricing Calculator



\* Supports multiple steel types:



&#x20; \* HRS (Hot Rolled Steel)

&#x20; \* CR (Cold Rolled Steel)

&#x20; \* HDG (Hot Dip Galvanized)

\* Automatic calculation of:



&#x20; \* Mill surcharges

&#x20; \* SSC (processing) costs

&#x20; \* Margin

&#x20; \* Transport and additional costs

\* Real-time updates



\---



\### 📄 Offer Management



\* Add items to an offer summary

\* View all positions in one place

\* Automatic total value calculation

\* Edit, duplicate, and delete entries



\---



\### 📑 PDF Export



\* Generate offers in PDF format

\* Ready to send to clients or supervisors
